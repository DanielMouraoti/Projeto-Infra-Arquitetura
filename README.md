# 🌐 Projeto de Infraestrutura Física Corporativa

Este repositório contém a documentação da infraestrutura física de rede de uma unidade corporativa. O objetivo é fornecer uma visão clara da distribuição de ativos, setores e estações de trabalho.

## 🚀 Estrutura do Projeto

O diagrama foi desenvolvido utilizando o **draw.io** e foca nos seguintes pilares:
* [cite_start]**Segmentação por Departamentos**: Divisão visual entre Infraestrutura, Desenvolvimento, Negócios, Administrativo e Diretoria[cite: 160, 161, 162, 163, 164].
* [cite_start]**Segurança e Anonimato**: Todos os nomes de colaboradores reais foram substituídos por figuras históricas da ciência (ex: Newton, Tesla, Lovelace) para fins de portfólio e privacidade[cite: 101, 102, 143].
* [cite_start]**Redundância de Link**: Mapeamento de dois provedores de internet (Vivo e NET) para garantir alta disponibilidade[cite: 168, 169].
* [cite_start]**Gestão de Ativos**: Identificação única de cada estação através de IDs de usuário (ex: Usr19, Usr06)[cite: 101, 143].

## 🛠️ Ativos Mapeados
* [cite_start]**Switches Layer 2**: Interconexão dos dispositivos finais nos setores[cite: 151, 152].
* [cite_start]**Servidor Dedicado**: Centralização de serviços e dados da unidade[cite: 149, 156].
* [cite_start]**Modems/Gateways**: Terminação dos links de fibra óptica[cite: 155, 158].
* [cite_start]**Periféricos**: Impressoras e estações rotativas para estagiários[cite: 166, 133].
